Linux Commands List

Master the essential Linux commands to navigate and control your system like a pro.

ls

List directory contents. Use ls -la to see all files, including hidden ones, with detailed information.

cd

Change the current directory. Use cd /path/to/directory to navigate to a specific folder.

pwd

Print the current working directory. Useful to know where you are in the filesystem.

mkdir

Create a new directory. Use mkdir foldername to create a new folder.

rm

Remove files or directories. Use rm -r directoryname to delete a directory and its contents.

cp

Copy files or directories. Use cp source destination to copy files to another location.

mv

Move or rename files. Use mv oldname newname to rename, or mv filename /new/path/ to move files.

cat

Concatenate and display file content. Use cat filename to view the content of a file.

chmod

Change file permissions. Use chmod 755 filename to modify permissions for a file.

chown

Change file owner and group. Use chown user:group filename to change the ownership of a file.

Master these commands to become a Linux expert!
