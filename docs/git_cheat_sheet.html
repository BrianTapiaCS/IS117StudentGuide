Git Cheat Sheet

Basic Git Commands

  • git init - Initialize a new Git repository.
  • git clone [url] - Clone an existing repository from a URL.
  • git add [file] - Stage a file for the next commit.
  • git commit -m "message" - Commit staged changes with a descriptive message.
  • git status - Show the status of the working directory and staging area.
  • git log - Show the commit history for the repository.
  • git push - Push local commits to the remote repository.
  • git pull - Fetch and merge changes from the remote repository.
  • git branch - List all branches in the repository.
  • git checkout [branch] - Switch to a specified branch.
  • git merge [branch] - Merge a branch into the current branch.
  • git remote -v - List the remote connections.

Undoing Changes

  • git reset [file] - Unstage a file, keeping the changes in the working directory.
  • git checkout -- [file] - Discard changes in the working directory.
  • git revert [commit] - Create a new commit that undoes the changes from a specified commit.
