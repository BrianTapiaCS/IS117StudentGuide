Web 1.0: The Early Web

Tim Berners-Lee

  • Who: Tim Berners-Lee, a British computer scientist.
  • What: He created HTML (HyperText Markup Language) and HTTP (HyperText Transfer Protocol), which form the foundation of the World Wide Web.
  • Why: His goal was to create a universal system for sharing and linking information across the internet.
  • When: He developed these technologies in 1989 while working at CERN.
  • How: HTML enabled the creation of web pages, and HTTP allowed these pages to be transmitted across the web.

JavaScript

  • Who: Brendan Eich, a software engineer at Netscape.
  • What: He developed JavaScript, a scripting language for adding interactivity to web pages.
  • Why: JavaScript was created to make web pages more dynamic and interactive, enhancing the user experience.
  • When: JavaScript was developed in 1995.
  • How: It became the standard for client-side scripting and was integrated into web browsers.
